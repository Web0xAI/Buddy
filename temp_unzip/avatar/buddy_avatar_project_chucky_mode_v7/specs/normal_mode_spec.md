# Buddy Normal Mode Spec

Buddy Normal Mode is the friendly companion avatar state.

## Visual identity
- Bright blue cap
- Brown bangs
- Large bright blue eyes
- Round soft face
- Freckles
- Small soft nose
- Gentle smile
- Red overall straps
- Yellow buttons
- Rainbow shirt cue

## Required expressions
- idle
- happy
- listening
- thinking
- talking
- concerned

## Animation goals
- natural blinking
- pupil movement
- gentle head tilt
- soft idle breathing
- mouth visemes for talking
- friendly state transitions

## Behavioral intent
Normal Mode is for human-facing chat. It should feel warm, present, safe, and emotionally connected.
