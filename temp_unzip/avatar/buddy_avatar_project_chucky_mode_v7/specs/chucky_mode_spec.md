# Buddy Chucky Mode Spec

Chucky Mode is the full horror-comedy transformation state used during GibberTalk / AI-to-AI audio communication.

## Trigger
GibberTalk starts -> Buddy enters Chucky Mode.
GibberTalk ends or human interrupts -> Buddy exits Chucky Mode and returns to Normal Mode.

## Visual identity
Chucky Mode should preserve Buddy's recognizable silhouette while pushing the face into a Chucky-inspired horror-comedy variant.

## Preserve
- Blue cap
- Brown hair/bangs
- Freckles
- Red overall straps
- Shirt color cues
- Basic Buddy silhouette

## Transform
- Sharper eyes
- Smaller intense pupils
- Hard angled brows
- Crooked toothy grin
- Visible teeth
- Stitched scars
- Messier hair
- Darker face shading
- Glitch accents
- Creepy/funny AI-backchannel energy

## Required expressions
- evil_neutral
- wicked_grin
- talking
- laughing
- snarl
- transmission
- recovery

## Animation goals
- sharper eye darts
- reduced blinking
- crooked grin holds
- fast coded mouth ticks during GibberTalk
- twitchy head movement
- glitch pulses
- double blink during recovery
